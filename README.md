## Giới Thiệu 

Một quyển sách mã nguồn mở được viết với [PreTeXt](https://pretextbook.org/index.html).

Giai đoạn hiện tại: Alpha

**Triển khai hiện tại:** [Github Pages](https://vinhphmcng.github.io/ma-tran-so-cap/)

### Tầm Nhìn & Mong Muốn
